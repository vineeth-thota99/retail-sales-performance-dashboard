# Retail Sales Performance Dashboard

## Tools Used
Power BI, Excel

## Objective
Analyze retail sales trends, regional performance, and category KPIs.

## Key Metrics
- Total Sales
- Orders
- Customers
- Regional Performance

## Key Insights
- Strongest sales region identified
- Top performing categories highlighted
- Seasonal trends visualized
